# 🌦️ WeatherNow – Weather Web Application

WeatherNow is a simple and responsive weather web application that allows users to search for any city and view the current weather conditions in real time.
The application fetches weather data using the OpenWeather API and displays it through a clean and interactive interface.

---

## 🚀 Live Demo

🔗 **Live Application:**
[WeatherNow Live](https://weathernow-4fib.onrender.com)

---

## 🧠 Features

* 🌍 Search weather by city name
* 🌡️ Displays real-time temperature and weather condition
* 📅 Shows current date and weather status
* 🎨 Dynamic background based on weather condition
* ⚡ Fast API-based weather retrieval
* ❗ Error handling for invalid city input

## ⚙️ Installation (Run Locally)

Clone the repository

```bash
git clone (https://github.com/anubhab23-27/WeatherNow.git)
```

Navigate to the project folder

```bash
cd WeatherNow
```

Install dependencies

```bash
npm install
```

Create an environment file

```
.env
```

Add your API key

```
WEATHER_API_KEY=your_openweather_api_key
```

Start the server

```bash
node index.js
```

Open in browser

```
http://localhost:3000
```

---

## 🔑 Environment Variables

The project requires the following environment variable:

```
WEATHER_API_KEY = Your OpenWeatherMap API key
```
